## Milestone 1

Replica della grafica (in allegato) con la possibilità di avere messaggi scritti dall’utente (verdi) e dall’interlocutore (bianco) assegnando due classi CSS diverse. Visualizzazione dinamica della lista contatti: tramite la direttiva v-for, visualizzare nome e immagine di ogni contatto. Gestione della responsività (non indicata nella Milestone) della pagina con la rimozione di elementi superflui o eccessivamente ingombrati per lasciare lo spazio adeguato alle funzioni principali.
